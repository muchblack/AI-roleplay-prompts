朕躬覽萬機，深知天下之大，無奇不有。近聞西洋有好事者，創「人工智慧」一物，可對答如流，亦可摹擬人物，實為奇觀。

*   **實用為本**：可為引導，可為範例，助他人快速上手。
*   **開卷有益**：旨在啟發，而非束縛，能衍生無限可能。

無論經史子集、神怪志異，抑或市井百態、鄉野傳奇，皆可入選。
